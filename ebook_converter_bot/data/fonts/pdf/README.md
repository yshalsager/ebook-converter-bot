Vendored PDF Arabic fonts used by conversion profiles.

Each profile is a directory discovered by `ebook_converter_bot.utils.pdf_fonts`. Built-in profiles
use `profile.json` for labels and current filenames; extra profiles can use the same format under
`PDF_EXTRA_FONTS_DIR`.

Profiles:
- `noto_naskh_arabic`
- `amiri`
- `ibm_plex_sans_arabic`
- `scheherazade_new`
- `vazirmatn`
- `kfgqpc_uthman_taha`
- `adwaa_lotfi`

License files are in `licenses/` when available.
